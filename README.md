# Менеджер подписок

Приложение для управления клиентами на подписке с календарём продлений, историей платежей и email-уведомлениями.

## Возможности

- 📅 **Календарь продлений** — визуальное отображение дат окончания подписок
- 👥 **База клиентов** — полная информация о каждом клиенте
- 💳 **История платежей** — все транзакции с фильтрацией
- 📧 **Email-уведомления** — напоминания о продлении через mailto
- 📊 **Статистика** — MRR, выручка, количество клиентов
- 💾 **Сохранение данных** — данные хранятся в браузере (localStorage)
- 📤 **Экспорт/Импорт** — резервное копирование в JSON

## 🚀 Развёртывание на Vercel (рекомендуется)

### Вариант 1: Через GitHub (автодеплой)

1. Создайте репозиторий на GitHub
2. Загрузите туда эти файлы
3. Перейдите на [vercel.com](https://vercel.com)
4. Нажмите "New Project"
5. Импортируйте репозиторий из GitHub
6. Vercel автоматически определит настройки
7. Нажмите "Deploy"

Готово! Ваше приложение будет доступно по адресу типа `https://subscription-manager-xxx.vercel.app`

### Вариант 2: Через Vercel CLI

```bash
# Установите Vercel CLI
npm install -g vercel

# Войдите в аккаунт
vercel login

# Разверните проект
vercel

# Для продакшена
vercel --prod
```

## 🌐 Развёртывание на Netlify

1. Перейдите на [netlify.com](https://netlify.com)
2. Нажмите "Add new site" → "Deploy manually"
3. Сначала соберите проект локально:

```bash
npm install
npm run build
```

4. Перетащите папку `dist` в окно Netlify
5. Готово!

Или через GitHub:
1. Подключите репозиторий
2. Build command: `npm run build`
3. Publish directory: `dist`

## 📧 Настройка email-уведомлений

Приложение использует `mailto:` ссылки для отправки писем. При нажатии на кнопку "Напомнить":

1. Открывается ваш почтовый клиент (Outlook, Gmail и т.д.)
2. Письмо уже заполнено по шаблону
3. Копия отправляется на email администратора
4. Вам остаётся только нажать "Отправить"

Для автоматической отправки писем потребуется серверная часть (например, Vercel Functions + SendGrid).

## 💾 Хранение данных

Данные сохраняются в localStorage браузера:
- `subscription_clients` — список клиентов
- `subscription_payments` — история платежей
- `subscription_notifications` — журнал уведомлений
- `subscription_settings` — настройки приложения

Используйте функцию Экспорт/Импорт в настройках для резервного копирования.
